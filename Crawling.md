# Node.js Base Web Project

## 6. Crawling

### 6.1 Crawling 개요
- 크롤링 순서
    - URL에서 HTML을 가져온다.
    - 내가 원하는 부분을 가져온다.
- 패키지 다운로드
    - requset : url로 호출 후 데이터를 가져오고 
    - npm install request
    - npm install request-promise
    - cheerio : jQuery방식으로 DOM을 가져와서 데이터를 정제한다.
    - npm install cheerio
